Microsoft Azure App Registration

You need a Microsoft Azure app registration to access the OneNote API.

üìã How to create an Azure App Registration
  1. Go to Microsoft Entra admin center
  2. Navigate to Applications ‚Üí App registrations
  3. Click New registration
  4. Enter a name (e.g., "OneNote Exporter")
  5. Set Supported account types to "Personal Microsoft accounts only"
  6. Set Redirect URI to: http://localhost:8080/auth/callback
  7. Click Register
  8. Copy the Application (client) ID
  9. Go to Certificates & secrets ‚Üí New client secret
  10. Copy the secret Value (not the ID)
  11. Go to API permissions ‚Üí Add a permission
  12. Add these Microsoft Graph permissions:
    • Notes.Read
    • Notes.Read.All
    • User.Read
Your current secret is saved. Leave blank to keep existing.
